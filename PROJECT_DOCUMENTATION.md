## 📱 Введение

**Fresh Market** - это полнофункциональная платформа электронной коммерции, созданная с использованием современных веб-технологий. Платформа предоставляет три различных интерфейса:

- **Для покупателей** - просмотр товаров, поиск, покупка и отслеживание заказов
- **Для поставщиков** - управление каталогом товаров, обработка заказов и просмотр аналитики
- **Для администраторов** - верификация поставщиков, мониторинг системы и управление пользователями

│

### Хранилище Данных

Приложение использует **localStorage** для хранения данных:

```javascript
// Пользователи
localStorage.setItem("users", JSON.stringify([...users]))

// Товары поставщиков
localStorage.setItem("supplierProducts", JSON.stringify([...products]))

// Заказы
localStorage.setItem("orders", JSON.stringify([...orders]))

// Текущий пользователь
localStorage.setItem("user", JSON.stringify(currentUser))
```

### Процесс Оформления Заказа

```
Добавление в корзину
        ↓
Просмотр корзины
        ↓
Оформление заказа
        ↓
Заполнение адреса доставки
        ↓
Подтверждение заказа
        ↓
Сохранение в localStorage
        ↓
Переход в "Мои заказы"
```

## 🚀 Инструкция по установке

### Требования
- Node.js 16+
- npm или yarn

### Установка

1. **Перейдите в директорию проекта:**
```bash
cd fresh-market
```

2. **Установите зависимости:**
```bash
npm install
```

3. **Запустите dev-сервер:**
```bash
npm run dev
```

4. **Откройте браузер:**
```
http://localhost:5173
```

### Сборка для продакшена

```bash
npm run build
```

## 📖 Использование платформы

### Путь Покупателя

1. **Регистрация:**
   - Нажмите "Регистрация" в навбаре
   - Выберите "Покупатель"
   - Заполните форму
   - Нажмите "Создать аккаунт"

2. **Просмотр Товаров:**
   - Перейдите в "Магазин"
   - Используйте поиск или категории для фильтрации
   - Сортируйте по цене или рейтингу

3. **Добавление в Корзину:**
   - Нажмите на товар для просмотра деталей
   - Нажмите "Добавить в корзину"
   - Товар появится в корзине (иконка 🛒)

4. **Оформление Заказа:**
   - Откройте корзину (🛒)
   - Проверьте товары и цену
   - Нажмите "Оформить заказ"
   - Введите адрес доставки
   - Подтвердите заказ

5. **Просмотр Заказов:**
   - Нажмите на аватар (правый верхний угол)
   - Выберите "Мои заказы"
   - Просмотрите статусы заказов

### Путь Поставщика

1. **Регистрация:**
   - Нажмите "Регистрация"
   - Выберите "Поставщик"
   - Заполните данные магазина
   - Создайте аккаунт

2. **Добавление Товаров:**
   - Перейдите на "Панель управления" (аватар)
   - Вкладка "Товары"
   - Заполните форму и нажмите "Добавить товар"

3. **Обработка Заказов:**
   - Во вкладке "Заказы"
   - Просмотрите входящие заказы
   - Измените статус (Подтвердить → Отправить → Доставлено)

4. **Просмотр Аналитики:**
   - Во вкладке "Обзор"
   - Просмотрите статистику магазина

### Путь Администратора

1. **Вход:**
   - Email: admin@demo.com
   - Пароль: demo123

2. **Верификация Поставщиков:**
   - Откройте "Администрация" (аватар)
   - Вкладка "Поставщики"
   - Нажмите "Одобрить" или "Отклонить"

3. **Мониторинг Системы:**
   - Вкладка "Обзор" - статистика
   - Вкладка "Пользователи" - база пользователей
   - Вкладка "Заказы" - история заказов

## 🐛 Возможные Проблемы и Решения

### Проблема: Пустая корзина после перезагрузки
**Решение:** Данные корзины хранятся в React state. Реализуйте сохранение в localStorage:
```javascript
useEffect(() => {
  localStorage.setItem("cart", JSON.stringify(items));
}, [items]);
```

### Проблема: Данные не сохраняются
**Решение:** Проверьте, включены ли localStorage в браузере (не в инкогнито режиме)

### Проблема: Стили не загружаются
**Решение:** Убедитесь, что пути к CSS файлам корректны в импортах
